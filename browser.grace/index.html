So anyway... back to the browser: the browser is just like any other browser atm, you can browse any website, open 20 tabs if you want, and even print pages. And you can still pretend that you're "hacking" with Cmd+Chift+C just like on Chrome.

The browser also has compatibility with what I'm calling Grace Sites, which are essentially a set of websites with the ".grace" web extension, which can only be accessed using the Grace browser. Currently we only have browser.grace, but I'm working on adding more like news.grace, and even my own personal website, gabriel.grace.

There's also some pretty cool auto-update software that I wrote myself in the browser, which is pretty cool; I was thinking of actually making the auto-update software open source and calling it "The Golden Grace Bridge" (ikr the name is SO cringe) but I decided not to.

The browser is called Grace because of two people: Grace Hopper (basically the Einstein of CompSci), and the other Grace (who I'm just not going to refer to by full name because... yeah. no.).

So the browser is in memory of Grace Hopper, and for the other Grace.
